# Dijkstra Algorithm

This project is an application written in C++ that finds shortest paths using Dijkstra algorithm. It works on a graph with 9 nodes in total.

## Starting Point
Home (node 0)

## Tools Used
- C++
- Dev-C++
- Git, GitHub
